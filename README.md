# Translasi PvZ Fusion Bahasa Indonesia

### Cara Pasang Mod
1. Download file PvZFusionIDTranslate.dll dan isi dari
2. Download [MelonLoader](https://melonloader.co/)
3. Buka file exe dan pilih file exe game
4. Tunggu sampai selesai dan masukkan file yang di download (kecuali MelonLoader) pada folder Mods
5. Selamat Bermain!

### Fitur
**Font Switcher (F6): ** Menukar font (termasuk font original game). 
**Reload (F5): ** Reload mod. 
**Toggle FPS (F4): ** Tampilkan/Sembunyikan FPS (mengikuti font yang di pilih)
**Update Notifier: ** yang akan muncul di log game, semacam pemberitahuan bahwa ada versi baru game atau mod yang tersedia. 
**Leak** untuk update selanjutnya
**Kecepatan Load Mod: **140ms (0.14 detik) - 2000 ms (2 detik)
