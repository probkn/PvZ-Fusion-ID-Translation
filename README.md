# PvZ Fusion Bahasa Indonesia

Mod translasi untuk game **Plants vs. Zombies Fusion**. Mod ini bertujuan untuk memudahkan pemain Indonesia memahami alur permainan dan fitur-fitur di dalam game.
Mod ini sudah tersedia untuk versi 3.5 dan beberapa versi kebawah
---

### 🛠️ Cara Pasang Mod
1. **Download** file `PvZFusionIDTranslate.dll` dari bagian [Releases](https://github.com/probkn/PvZ-Fusion-ID-Translation/releases).
2. **Download & Install** [MelonLoader](https://melonloader.co/).
3. Buka **MelonLoader.Installer.exe**, pilih file `.exe` game PvZ Fusion kamu, lalu klik **Install**.
4. Jalankan game sekali, lalu tutup kembali untuk memunculkan folder `Mods`.
5. Masukkan file `PvZFusionIDTranslate.dll` ke dalam folder **Mods** di direktori game kamu.
6. **Selamat bermain!**

---

### ✨ Fitur Utama
* **Font Switcher (F6):** Menukar font secara real-time (termasuk font original game).
* **Reload Mod (F5):** Memuat ulang mod tanpa harus restart game.
* **Toggle FPS (F4):** Tampilkan atau sembunyikan FPS (tampilan mengikuti font yang dipilih).
* **Update Notifier:** Pemberitahuan otomatis di log game jika ada versi baru mod atau game.
* **Optimasi Tinggi:** Kecepatan load mod sangat cepat, hanya berkisar antara **140ms - 2000ms**.

---

### 👨‍💻 Fitur Developer (Debug Mode)
Gunakan tombol berikut untuk keperluan pengembangan dan debugging:

| Tombol | Fungsi | Keterangan |
| :--- | :--- | :--- |
| **F9** | **Dump Strings** | Mengambil semua teks mentah dari dalam game ke file eksternal. |
| **F10** | **Compare Dump** | Membandingkan hasil dump terbaru dengan file translasi yang ada untuk mengecek teks yang belum diterjemahkan. |
